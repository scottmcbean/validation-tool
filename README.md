# Validation Tool

Veris Coaching tool for partners (couples or business). Helps name the emotion + the friction, then teaches one sentence the user can carry into the next hard conversation, plus one forward question for next time.

**Live:** https://scottmcbean.github.io/validation-tool/

**Modes:**
- Default (couples): no query param needed
- Business partner: `?mode=business`

**Output:** Single-file HTML, no dependencies. Click "Save this to carry with you." to print a one-page pocket card (US Letter portrait, tri-fold friendly) with the user's filled sentence + forward question + scope-of-practice footnote.

## Build

`index.html` is generated by `build.py` (in `/tmp/validation-tool-build/` during dev) from a template + Marlowe's content lock. The build script copies the Söhne `@font-face` blocks verbatim from the Enneagram Sliders deployment so the paper-ink-signal aesthetic family stays consistent.
